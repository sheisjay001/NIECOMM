## Onboarding Goal

Help users quickly understand: - They can find gadgets **near them** -
Buying locally means **lower delivery costs** - Products are sold by
**trusted city vendors**

Target completion time: **30--45 seconds**

## Core UX Philosophy

-   Local-first messaging
-   Value before commitment
-   Minimal typing, tap-based choices

------------------------------------------------------------------------
